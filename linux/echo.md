`options`
```
-n: 뒤에 줄바꿈을 생략하고 출력을 표시합니다.
-E: 기본 옵션은 이스케이프 문자의 해석을 비활성화합니다.
-e: 다음 이스케이프 문자의 해석을 활성화합니다.
\\: 백슬래시 문자(\)를 표시합니다.
\a: 출력을 표시할 때 경고음을 재생합니다.
\b: 백 스페이스 키를 누르는 것과 동일한 백스페이스 문자를 생성합니다 .
\c: 이스케이프 문자 뒤에 오는 모든 출력을 생략합니다.
\e: Esc 키 를 누르는 것과 동일한 이스케이프 문자 입니다.
\f: 용지 공급 문자를 사용하면 프린터가 자동으로 다음 페이지의 시작 부분으로 이동합니다.
\n: 출력에 새 줄을 추가합니다.
\r: 캐리지 리턴을 수행합니다.
\t: 가로 탭 공간을 만듭니다.
\v: 세로 탭 공간을 만듭니다.
\NNN: 8진수 값이 NNN.
\xHH: 16진수 값이 HH.
#
```
