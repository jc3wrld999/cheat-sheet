#
`setup`

1. [download zip](https://github.com/NationalSecurityAgency/ghidra/releases)

2. unzip
```
unzip ghidra_*_PUBLIC_*.zip
```

3. openjdk 종속성 설치
```
apt-get install default-jdk
```

4. run
```
./ghidraRun
```
