enumlinux
#
`SMB 공유를 열거 (Samba)`

syntax

```
enum4linux <option> ip
```
options

**TAG**            **FUNCTION**

\-U             get userlist  
\-M             get machine list  
\-N             get namelist dump (different from -U and-M)  
\-S             get sharelist  
\-P             get password policy information  
\-G             get group and member list

\-a             위의 모든 것(전체 기본 열거)
