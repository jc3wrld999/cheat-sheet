#
`7zip`

설치
```
sudo apt-get install p7zip-full p7zip-rar
```

현재 디렉토리에 아카이브 생성
```
7z a <아카이브 파일 이름> <파일 목록>
```

아카이브 추출
```
7z e <아카이브 파일 이름>
```
